## SOS Emergency System Website

A web-based SOS Emergency System designed to provide quick access to emergency contacts and essential information during critical situations.

## Features
- Quick access to emergency services
- Simple and user-friendly interface
- Responsive design for mobile and desktop
- Awareness-focused emergency information

## Tech Stack
- HTML
- CSS
- JavaScript

## Purpose
The goal of this project is to improve emergency response awareness by providing a fast and accessible SOS interface.

## What I Learned
- Frontend web development fundamentals
- Designing accessible and responsive layouts
- Handling user interactions using JavaScript
- Building real-world problem-oriented applications
